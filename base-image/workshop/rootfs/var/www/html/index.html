Development repository download

Click here to download the development repository: repository.tar.gz

After downloading the repo, please run tar xvf repository.tar.gz to extract the data.

Docker image for development on laptops

Click here to download the Docker image: image.tar.gz

After downloading the image and the repository, run:

  1. cd roscon-2023-realtime-workshop
  2. docker/fetch path/to/repository.tar.gz
    • For example, if you downloaded the file to ~/Downloads, then run docker/fetch ~/Docker/repository.tar.gz.
